3dBREP

3D engine, to represent the 3D modeling from parametric solid B-rep.A simple 3D game engine written i AS3 (Flash ActionScript 3.0) . It is not commented or documented yet, and is an early version which is used for the specific project with the working for 'FRIEMO', which will not published until some time in 2013. The current code might be found useful, but requires some work on the readers part. Feel free to ask questions, and please do not write complaints without a constructive angle.
